About US

We love all cute fluffy friends! We started as a real life store and then decied to move online, note that we are not a real pets shop and are just a creative project for a web programming class. We designed this website to show off a bunch of cute animals using a bootstrap grid. It was a lot of fun and taught us a lot about web programming and how to use html and css.
